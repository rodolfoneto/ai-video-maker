# Credentials Format

## Algorithmia

File: `algorithmia.json`

```
{
    apiKey: ...
}
```

## IBM Watson

File: `watson-nlu.json`

```
{
    "apikey": "...",
    "iam_apikey_description": "...",
    "iam_apikey_name": "...",
    "iam_role_crn": "...",
    "iam_serviceid_crn": "...",
    "url": "..."
}
```
